## 3. 一页式执行顺序（先跑通）
1. 完成 Azure 资源与 deployment（按 `005`）。
2. 在 Cloud Shell 准备环境并安装依赖（按 `006`）。
3. 填三份 `.env`，确认 endpoint/key/索引名一致。
4. 跑 `create_RAG_index` notebook 建索引。
5. 先跑 `app-CRG/full_cot_with_tools.py` 作为 smoke test。
6. 通过后再扩到 `app-malt` 和 `app-traffic-analysis`。

当你的问题集中在 `text-embedding-ada-002`（比如“为什么看不到模型/如何单独验证 embedding”）时，建议按这个顺序读：
